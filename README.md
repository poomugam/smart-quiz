# smart-quiz

How to Start...

1. Make sure you have the nodejs installed.
2. Clone the repo:
    ```
    git clone https://github.com/poomugam/smart-quiz.git
    
    cd smart-quiz
    ```
3. Run the project.
    ```
      npm install
      gulp start 
      //or// gulp
      
    ```
4. Browser synnc automatically Open http://localhost:3000 in browser
